# Structured Markdown Notes

## Summary

This source-derived note argues that Markdown with consistent frontmatter is a durable and beginner-friendly format for building a knowledge base.

## Key Points

- Plain files reduce tooling lock-in.
- Frontmatter makes notes easier to search and organize.
- A file-first system can later be exported to richer tools.
