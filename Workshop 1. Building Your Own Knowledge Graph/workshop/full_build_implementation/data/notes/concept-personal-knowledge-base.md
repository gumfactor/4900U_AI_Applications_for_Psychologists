# Personal Knowledge Base

## Summary

A personal knowledge base is a structured collection of notes designed to preserve, connect, and reuse knowledge over time.

## Key Points

- Notes should be small and linkable.
- Structure should improve retrieval rather than add ceremony.
- The knowledge base becomes stronger when sources and links are visible.
