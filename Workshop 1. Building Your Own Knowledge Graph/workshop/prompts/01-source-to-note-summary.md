# Prompt: Source to Note Summary

Recommended model: `Gemini 2.5 Flash-Lite`

Use this prompt with one source at a time.

```text
You are helping build a personal knowledge base.

Read the source text below and produce a draft note in Markdown.

Requirements:
- Keep the output grounded only in the supplied source.
- Do not invent citations or facts.
- Write a one-paragraph summary.
- Provide 3-5 key points.
- Suggest 2-4 tags.
- Suggest up to 3 related note ideas.
- End with 2 open questions.

Output format:
- Summary
- Key Points
- Suggested Tags
- Suggested Related Notes
- Open Questions

Source text:
[PASTE SOURCE HERE]
```
