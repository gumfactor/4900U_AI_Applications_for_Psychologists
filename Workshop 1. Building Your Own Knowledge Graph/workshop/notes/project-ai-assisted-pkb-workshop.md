# AI-Assisted PKB Workshop

## Summary

This project uses a file-first knowledge base in VS Code and adds Gemini as a controlled layer for drafting, extraction, linking, and question answering.

## Key Points

- The workshop succeeds even if AI setup is incomplete.
- Gemini 2.5 Flash-Lite is the default model for free-tier-friendly classroom tasks.
- Human review and Git history are part of the system, not optional extras.
