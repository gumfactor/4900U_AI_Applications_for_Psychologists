# Structured Markdown Notes

## Summary

Structured Markdown notes combine human readability with just enough metadata for filtering, automation, and long-term portability.

## Key Points

- Plain text keeps the system durable.
- Frontmatter creates a machine-readable contract.
- Version control works naturally with text files.
