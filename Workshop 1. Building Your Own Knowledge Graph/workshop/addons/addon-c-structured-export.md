# Addon C: Structured Export

Export note metadata and links to `JSON` or `CSV` so students can inspect the note collection as graph-shaped data.

Suggested export fields:

- `id`
- `title`
- `type`
- `status`
- `tags`
- `source_refs`

Suggested link export fields:

- `source_id`
- `relationship`
- `target_id`
