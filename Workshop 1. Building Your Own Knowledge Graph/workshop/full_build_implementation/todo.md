# Feature Todo

This file organizes promising next features for the knowledge graph app and adds simple status tracking.

| Feature | Priority | Percent Complete | Done Status | Notes |
| --- | --- | ---: | --- | --- |
| Due Dates and Reminder Views | High | 100% | Done | Added optional due dates for notes plus reminder-focused `Due Today`, `Upcoming`, and `Overdue` browser views. |
| Task Status Workflow | High | 100% | Done | Added persisted note statuses, notes-page status filtering, status badges, and quick status updates from the note detail view. |
| Saved Searches / Smart Collections | Medium | 0% | Todo | Let users save reusable filter combinations such as reminder-driven or topic-specific note collections. |
| Backlinks and Mention Discovery | Medium | 0% | Todo | Surface notes that mention or reference the current note, even when links were not manually added. |
| AI Link Suggestions on Save | Stretch | 0% | Todo | Suggest related notes during note creation so the graph grows with less manual linking effort. |

## Status Legend

- `Percent Complete`: rough implementation progress for the feature as a whole.
- `Done Status`: use `Todo`, `In Progress`, `Blocked`, or `Done`.
