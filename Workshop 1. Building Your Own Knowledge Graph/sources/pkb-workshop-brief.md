# Workshop Source Brief

A personal knowledge base works best when notes are small, clearly typed, and linked to other notes. The system should support both storage and thinking. Good note systems distinguish raw capture from reviewed knowledge, keep source provenance visible, and make retrieval easy through search, links, and metadata.

AI can help summarize material, propose metadata, and suggest note connections. However, the user should keep the notes themselves as the source of truth. AI-generated drafts should be reviewed before they are treated as final knowledge.

In a beginner-friendly workflow, the best starting point is usually plain Markdown files with consistent frontmatter and a small set of note types. This creates a durable system that can later be exported to databases, graph tools, or retrieval systems.
