# Personal Knowledge Base

## Summary

A personal knowledge base is a structured collection of notes designed to help a person store, connect, retrieve, and reuse knowledge over time.

## Key Points

- The system should support retrieval as well as storage.
- A useful PKB relies on consistent note structure.
- Linking notes turns isolated documents into a graph of ideas.
