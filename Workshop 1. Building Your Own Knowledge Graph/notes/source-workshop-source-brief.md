# Workshop Source Brief

## Summary

The workshop source brief argues for a beginner-friendly, file-first knowledge base that stays human-readable while supporting AI-assisted drafting and retrieval.

## Key Points

- Notes should be small, typed, and linked.
- AI is useful for enrichment, not for replacing source-truth notes.
- Markdown with frontmatter is a practical starting point for beginners.
