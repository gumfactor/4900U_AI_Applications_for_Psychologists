# Knowledge Graph Thinking

## Summary

A note collection behaves like a small knowledge graph when each note records meaningful relationships to other notes rather than existing in isolation.

## Key Points

- Relationship labels explain why a connection matters.
- Graph structure can emerge from plain files.
- Search and links work best together.
